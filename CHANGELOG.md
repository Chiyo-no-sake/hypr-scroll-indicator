# Changelog

## 0.1.0 — 2026-04-04

Initial release.

- Browser-like scrollbar overlay for Hyprland scrolling layout
- Smooth bezier easing matching Hyprland workspace animations
- Configurable colors via CLI or Hyprland-format colors file
- Auto-hides when window tape fits in viewport
- Monitor hot-plug detection
- Shell wrapper for automatic LD_PRELOAD detection
